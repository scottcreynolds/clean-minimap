KaizenMinimap is a simple, light-weight World of Warcraft enabling my opinion of a clean and minimalist minimap.

There is no in-game config (presently, not sure I will ever add it). The options are set with bool values from within KaizenMiniMap.lua. Feel free to set them to match your own opinions of what a clean minimap looks like.

Defaults (Feel free to edit and reload UI if your preferences don't match mine. They are at the top of the file)

local showWorldMapButton = false 				--hides the world map button
local showZoomButtons = false		 				--hides the zoom in/out buttons
local showCalendar = true				 				--calender shown by default
local showTracking = true				 				--tracking shown by default
local showZoneText = true				 				--zone text shown
local showZoneTextBorder = false 				--ugly border around zone text hidden
local showMinimapToggleButton = false 	--minimap toggle button hidden
